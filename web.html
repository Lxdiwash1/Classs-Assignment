BSc Web Development

Create modern websites and applications

Course Overview

Web Development teaches how to build websites and web applications using modern technologies. It covers front-end development, back-end programming, and full-stack solutions.

Students learn to create responsive, user-friendly, and secure web applications. These skills prepare them for careers in web design, web development, and digital product creation.

Main Subjects

  • HTML & CSS
  • JavaScript
  • Responsive Web Design
  • Web Applications
